01  가정법 과거
★★☆ 정답 ①
If I knew her phone number, I would call her right away.

가정법 과거는 현재 사실과 반대되는 상황을 가정할 때 사용한다. If절에 동사의 과거형, 주절에 would / could / might + 동사원형을 쓴다.

공식 If + S + 과거동사 ~, S + would / could + V …

오답 분석

  • ② will call — 직설법 미래, 현실 가정에 사용 불가
  • ③ called — 과거시제, 주절 동사로 단독 사용 불가
  • ④ would have called — 가정법 과거완료 주절형, 시제 불일치
02  가정법 과거완료
★★★ 정답 ②
If she had heeded the warning, the accident would not have happened.

가정법 과거완료는 과거 사실과 반대되는 상황을 가정한다. If절에 had + p.p., 주절에 would have + p.p.를 사용한다. heed는 "주의를 기울이다"의 뜻으로, 그녀가 경고를 따랐다면 사고가 나지 않았을 것임을 나타낸다.

공식 If + S + had + p.p. ~, S + would have + p.p. …

오답 분석

  • ① heeded — 단순 과거, 가정법 과거완료 If절에 부적합
  • ③ would heed — 주절 형태, If절에 사용 불가
  • ④ has heeded — 현재완료, 가정법 구조에 맞지 않음
03  혼합가정법
★★★ 정답 ①
If he had taken the medicine last night, he would feel much better now.

혼합가정법은 If절(과거 사실의 반대)과 주절(현재 사실의 반대)의 시제가 다를 때 사용한다. now, today 등 현재 시점 부사가 주절에 있으면 주절을 would + V(현재)로 쓴다.

공식 If + S + had + p.p. ~, S + would + V … (now / today)

오답 분석

  • ② would have felt — 주절이 과거 기준, now와 충돌
  • ③ had felt — If절 형태, 주절에 사용 불가
  • ④ feels — 직설법 현재, 가정법 주절 구조가 아님
