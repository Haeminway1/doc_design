The "Ship of Theseus" is a thought experiment that raises profound questions about identity and persistence over time. The paradox asks whether an object that has had all of its components replaced remains fundamentally the same object. In the classic formulation, the ship of the hero Theseus is preserved by the Athenians, who gradually replace its old, decaying planks with new ones. ( ① ) Eventually, no original plank remains. ( ② ) The question then arises: is this restored ship still the same Ship of Theseus? ( ③ ) To complicate matters, imagine that someone collected all the old planks and reassembled them. ( ④ ) Now there are two ships, and one must ask which, if either, is the true Ship of Theseus. ( ⑤ ) This paradox has a simple and universally accepted solution based on an object's function.
