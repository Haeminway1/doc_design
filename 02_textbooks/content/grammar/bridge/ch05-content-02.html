Focus 1: 원급, 비교급, 최상급의 기본 형성 원리 II
2. 음절에 따른 규칙적 변화

대부분의 형용사와 부사는 음절 수에 따라 다음과 같은 규칙으로 비교급과 최상급 형태를 만듭니다.

구분비교급 (-er / more)최상급 (-est / most)
1음절 단어
(big, fast, hot)		 원급 + -er
fast → faster		 원급 + -est
fast → fastest
2음절 단어
(-y, -er, -ow, -le 로 끝나는 경우)		 원급 + -er
happy → happier, clever → cleverer		 원급 + -est
happy → happiest, clever → cleverest
2음절 이상 단어
(famous, beautiful, expensive)		 more + 원급
famous → more famous		 most + 원급
famous → most famous
Vera's Tip | 철자 변화에 주의하세요!

-er이나 -est를 붙일 때, 단어의 끝 철자에 따라 다음과 같은 변화가 일어납니다.

  • 자음 + y로 끝나는 경우: y를 i로 바꾸고 붙입니다. (e.g., happy → happier, easy → easiest)
  • 단모음 + 단자음으로 끝나는 경우: 마지막 자음을 한 번 더 씁니다. (e.g., big → bigger, hot → hottest)
  • e로 끝나는 경우: r 또는 st만 붙입니다. (e.g., large → larger, wise → wisest)
