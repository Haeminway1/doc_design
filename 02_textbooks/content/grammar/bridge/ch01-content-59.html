4.3.2. 당위 형용사 (Mandative Adjectives)

"~해야 한다"는 의무·필요·요구·제안의 의미를 담아 `that`절에서 동사원형(`(should) + R`)을 요구하는 형용사입니다. 이는 가정법의 일종으로, 사실을 서술하는 직설법과 구별됩니다.

기본 구조: It + be + 당위 형용사 + that + S + (should) + 동사원형

  • `should`는 생략 가능하며, 의미 변화는 없습니다.
  • `that`절의 동사는 주어의 인칭이나 수에 관계없이 항상 동사원형을 사용합니다.
  • 수동태일 경우 `be + p.p.` 형태를 사용합니다. (e.g., It is vital that the plan be approved.)
의미암기 필수 형용사 목록
의무/필수necessary, essential, mandatory, compulsory, required
제안/권고advisable, desirable, recommended
적절/마땅appropriate, proper, fitting, natural
긴급/중요important, vital, crucial, imperative, urgent, critical

예시: It is vital that everyone (should) be present. / It is imperative that the data (should) be analyzed immediately.

변형 패턴 비교
형태구조예문
It ~ that S V It + be + adj + that + S + (should) + V It is essential that he (should) leave now.
for + 목적어 + to V It + be + adj + for + O + to V It is essential for him to leave now.
명사형 사용 It + be + N + that S V It is a requirement that every member attend.
