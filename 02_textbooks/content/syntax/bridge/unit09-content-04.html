Testing Point 28: be used to-v vs. be used to v-ing

전체 기출 빈도: ★★☆☆☆ 최근 기출 빈도: ★★☆☆☆

핵심 개념: '목적'과 '익숙함'의 의미 차이 구분
`used`와 관련된 세 가지 표현은 형태가 비슷하지만 의미가 완전히 다르므로, 주어와 문맥을 통해 정확히 구분해야 합니다.

'used'의 세 가지 얼굴

표현 주어 의미
used to + 동사원형 사람/사물 ~하곤 했다 (과거의 습관/상태)
be used to + 동사원형 사물 ~하는 데 사용되다 (수동태, 목적)
be used to + v-ing/명사 사람 ~하는 데 익숙하다 (상태, 습관)

예제 분석: The Masai don't slaughter their cattle for food; but if a cow is killed, then the hides are used to make / making shoes, clothing, and bed coverings.

1. 주어 확인: `the hides` (가죽) → 사물.
2. 의미 파악: 가죽이 신발을 '만드는 데 사용되다'는 목적/용도를 나타냄.
3. 구문 선택: `be used to + 동사원형` 구문.
4. 결론: 동사원형 make가 정답.
