## 3. Part 4 — 어법 선택형 연습

### 3-1. 지시문
> **괄호 안에서 어법상 쓰임이 바른 것을 고르시오.**

### 3-2. 문제 구조
- **형식:** 한국어 번역 먼저 제시 → 아래에 영어 문장 (오류 후보를 `[A / B]` 괄호로 표시)
- **번호 표기:** 문장 끝 위첨자 번호 (예: `Many students learn new languages [because / because of] school requirements.¹⁾`)
- **괄호 개수:** 문장당 1~5개 (평균 2~3개)
- **선택지 유형:** 항상 2지선다 (`[A / B]` 형태)
- **페이지 당 문항 수:** 4~10문항 (본문 길이에 따라 다름)

### 3-3. 문제 유형별 분류

#### A. 접속사/전치사 구분
```
Many students learn new languages [because / because of] school requirements.
→ because of (전치사 + 명사구)
```
```
It's not easy [because / because of] I don't know Spanish that well.
→ because (접속사 + 절)
```

#### B. to부정사 vs. 동명사
```
I want [understanding / to understand] interviews with my favorite players.
→ to understand (want + to부정사)
```
```
I suggest [watching / to watching] Spanish movies often.
→ watching (suggest + 동명사)
```

#### C. 현재분사 vs. 과거분사 (형용사 역할)
```
DREAM4 is back! I'm so [excited / exciting] [seeing / to see] my favorite Korean boy band perform.
→ excited / to see
```
```
We all know [what / that] a diet [contained / containing] a variety of foods keeps our bodies [healthy / healthily].
→ that / containing / healthy
```

#### D. 관계대명사 that vs. which vs. who
```
You should find friends [who / which] [is / are] [interested / interesting] in DREAM4 and [start / started] a club.
→ who / are / interested / start
```
```
Walnuts also have wrinkles, [that / which] the brain has too.
→ which (계속적 용법)
```

#### E. 주어-동사 수 일치
```
[Do / Doing] these things [is / are] fun and really [improve / improves] our Korean!
→ Doing / is / improves
```
```
Studies [show / shows] [that / what] walnuts help our brains [stay / staying] healthy and [active / actively].
→ show / that / stay / active
```

#### F. 동사 원형 vs. to부정사 (준사역/지각동사)
```
It will help you [get / getting] used to the sound of the language.
→ get (help + 목적어 + 동사원형)
```
```
It helps your eyes [process / processing] light and [send / to send] a clear image to the brain.
→ process / send
```

#### G. 부사 vs. 형용사
```
However, it's not [easy / easily] because I don't know Spanish that well.
→ easy (보어 위치 = 형용사)
```

#### H. 명령문 vs. 분사구문
```
[Follow / Following] DREAM4 on social media.
→ Follow (명령문)
```
```
[Look / Looking] at the following examples.
→ Look (명령문)
```
```
[Find / Finding] as many as you can and try to eat a variety of them.
→ Find (명령문)
```

#### I. 어순 (간접의문문)
```
They often post short messages in Korean about how [are they / they are] doing.
→ they are (간접의문문 = 평서문 어순)
```

#### J. 기타 (비교급 강조, 대명사, 수 표현)
```
Remember, every language is hard at first, but a new language can make your world [very / much] bigger!
→ much (비교급 강조)
```

## 4. Part 5 — 어색한 곳 찾기 연습

### 4-1. 지시문 (두 가지 변형)
> **I. 다음 글의 밑줄 친 부분 중 문맥상 어색한 것을 [N]개 찾아 바르게 고쳐 쓰시오.**  
> **II. 다음 글의 밑줄 친 부분 중 어법상 어색한 것을 [N]개 찾아 바르게 고쳐 쓰시오.**

- N = 2개 또는 3개 (섹션마다 고정)
- **I번(문맥)** 과 **II번(어법)** 이 한 페이지에 같이 출제됨 → 페이지당 2세트

### 4-2. 문제 구조
- **형식:** 본문 전체 또는 섹션 텍스트 블록 제시
  - I번(문맥): 오류 단어를 **굵은 볼드**로 표시
  - II번(어법): 오류 문장 전체를 **밑줄** 표시 (문장 단위 전체 밑줄)
- **답안 기입란:** `(1) __________ → __________ / (2) __________ → __________ / (3) __________ → __________`
- **화살표 형식:** 틀린 표현 → 바른 표현

### 4-3. 문맥 오류 유형 (I번)

#### A. 반의어 치환
```
"Why Do People Learn Native Languages?" → Native → Foreign
"it's not difficult because I don't know Spanish" → difficult → easy
"How can I impair my Spanish?" → impair → improve
```

#### B. 형용사 반의어
```
"keeps our bodies unhealthy" → unhealthy → healthy
"is also bad for that body part" → bad → good
"they look different" → different → similar
"help our brains stay healthy and inactive" → inactive → active
"eating tomatoes can raise your risk" → raise → lower
"if you want unhealthy eyes" → unhealthy → healthy
"which impairs your vision" → impairs → improves
```

#### C. 동사 반의어
```
"forget it" → forget → memorize
"Forget, every language is hard" → Forget → Remember
"Doing these things is fun and really spoils our Korean!" → spoils → improves
"which helps remove new, healthy cells" → remove → make
```

#### D. 부사 반의어
```
"you can understand the posts more difficultly" → difficultly → easily
```

#### E. 형용사 반의어 (맥락)
```
"become strange with the language first" → strange → familiar
"friends who are uninterested in DREAM4" → uninterested → interested
"their dancing are just imperfect" → imperfect → perfect
"If the people talk too slow" → slow → fast
"ginger's soft taste" → soft → strong
"ginger can be good for your heart" → heart → stomach
"make your world much smaller!" → smaller → bigger
```

### 4-4. 어법 오류 유형 (II번)

#### A. 단수/복수 오류

#### D. 관계사 오류

### 4-5. 페이지 구성 및 문항 수
- **1과 기준:** 본문 4개 섹션 × 2세트(문맥+어법) = **8세트**

### 5-1. 지시문

### 5-2. 문제 구조
- **상단:** 한국어 번역문 (완전한 문장)
- **중간:** 회색/노란 배경 박스 안에 **키워드 힌트** (쉼표로 구분된 단어들, 순서 고정)

- **특징:** 힌트 단어 순서가 고정 = 문법 구조를 암시하는 역할

### 5-3. 힌트 제공 방식

#### A. 핵심 어휘만 제공 (기본 조동사·관사 등은 빈칸)

```

#### B. 구문 힌트 포함 (숙어 단위)

→ "get used to"가 덩어리로 제공됨
```

#### C. 문장 일부 인쇄 + 나머지 빈칸
```
However, __________ __________ __________ because __________ __________ __________ __________ that well.
→ "However"와 "because"는 인쇄, 나머지는 빈칸
```

```
In any case, __ __ __ __ __ __ __ __ __ __. __ __ __ __ and __ __ __ __.
→ "In any case"와 "and"는 인쇄
```

## 8. 우리 워크북에 없는 문제 유형 (확장 가능 유형)

### 8-1. Part 5의 이중 세트 구조
- **현재 우리 워크북:** 단일 유형(문맥 OR 어법)
- **exam4u:** 같은 본문을 2회 제시 — I번(문맥 오류), II번(어법 오류)
- **확장 방법:** 동일 텍스트에 대해 문맥/어법 세트를 별도 출제

### 8-2. Part 5 II번의 "문장 전체 밑줄" 방식
- **현재 우리 워크북:** 특정 단어에만 밑줄
- **exam4u:** 오류를 포함한 문장 전체에 밑줄 → 어느 부분이 틀렸는지 학생이 직접 찾아야 함
- 난이도 높음, 변별력 있는 유형

### 8-3. 복수 오류 한 텍스트에서 찾기 (2개 vs. 3개)
- 본문이 짧은 섹션: 2개 오류
- 본문이 긴 섹션: 3개 오류
- 섹션 길이에 따라 오류 수 조정

### 8-4. 키워드 힌트 영작의 "구문 덩어리" 힌트
- 예: `get used to`, `not only...but also` 형태를 힌트 단어로 제공
- 구문 힌트 포함 문항은 문법 구조 학습을 명시적으로 유도

### 8-5. 힌트 없이 부분 제공 (scaffold 차등화)
- 일부 문항은 문장 프레임(접속사, 전치사 등)만 인쇄하고 핵심 부분을 빈칸 처리
- 예: `However, __ __ __ because __ __ __ __ that well.`

### 8-6. Answer Key의 "/" 분리 표기
- 전체 문장이 아닌 핵심 덩어리만 정답으로 제시
- 예: `I suggest watching Spanish movies / It will help you get used to the sound of the language.`

## 9. 400–500문항 확장을 위한 설계 가이드

### 9-1. 현재 exam4u 문항 밀도 분석
| Part | 1과 문항수 | 2과 문항수 | 페이지 수 |
|------|-----------|-----------|-----------|
| Part 4 | ~52 선택지 세트 | ~48 선택지 세트 | 4~5 |
| Part 5 | ~8 세트 (20~24 오류) | ~8 세트 (20 오류) | 4~5 |
| Part 7 | ~28 영작 문항 | ~27 영작 문항 | 6~7 |
| **합계(추정)** | **약 100문항** | **약 95문항** | **14~17p** |

### 9-2. 400–500문항 달성 전략

#### 전략 A: 세트 수 증가
- 각 본문 섹션당 Part 4 세트를 2~3세트로 늘림
  - 세트 1: 원문 그대로 (기본 2지선다)
  - 세트 2: 선택지를 3지선다로 확장
  - 세트 3: 더 많은 괄호 (문장당 4~5개)

#### 전략 B: Part 5 변형 확장
- 각 섹션에서 문맥 세트 × 2회 + 어법 세트 × 2회 = 4세트
- 오류 위치를 달리하여 동일 텍스트 재활용

#### 전략 C: Part 7 문항 세분화
- 현재: 1문장 = 1문항
- 확장: 복문을 분리하여 각 절을 별도 문항으로
- 또는: 같은 문장을 다른 키워드 힌트 조합으로 출제

#### 전략 D: 본문 섹션 수 증가
- 현재: 4개 섹션 (본문 1~4)
- 확장: 각 본문 내 단락을 더 잘게 나누어 6~8 섹션으로

#### 전략 E: Part 1~3·6 추가
- 아직 미확인된 Part 1~3(어휘, 문장 완성 등), Part 6(추정: 배열/영작 전단계)을 설계하여 추가

#### 전략 F: 난이도 3단계 분화
- 각 세트에 난이도 레이블: 기본 / 심화 / 실전
- 기본: 짧은 문장, 선택지 2개, 오류 1개
- 심화: 복문, 선택지 3개, 오류 2개
- 실전: 문장 전체 밑줄 방식, 오류 위치 미표시

## 10. 정리 요약

| 구분 | Part 0 | Part 4 | Part 5 | Part 7 |
|------|--------|--------|--------|--------|
| 핵심 능력 | 독해·번역 | 어법 판단 | 오류 식별·수정 | 영작 생성 |
| 문제 형식 | 없음(제시만) | [A/B] 이항 선택 | 밑줄+고쳐쓰기 | 키워드→문장 생성 |
| 답안 형식 | 없음 | 선택 | 틀린 표현→바른 표현 | 빈칸 채우기 |
| 문맥/어법 이중출제 | - | 없음 | 있음 (I·II번) | 없음 |
| 1과 총 문항 추정 | 0 | 28문장/52선택지 | 8세트/20+ | 28 |
| 2과 총 문항 추정 | 0 | 27문장/48선택지 | 8세트/20 | 27 |

**핵심 인사이트:**
1. exam4u는 **같은 본문을 7가지 다른 방식으로 반복** → 동일 문장이 Part 4·5·7에 모두 등장
2. Part 5는 **문맥 오류 + 어법 오류를 별도 세트**로 이중 출제 — 가장 효율적인 문항 증폭 방법
3. Part 7의 키워드 힌트는 **순서 고정** → 문법 구조를 유도하는 scaffold
4. 400~500문항을 만들려면 **Part 1~3·6 추가** + **각 파트 변형 세트 확장**이 핵심
