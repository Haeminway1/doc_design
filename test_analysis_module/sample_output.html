어법 영역 분석

03 어법 분석 (4/10, 40%)

영역 개요

어법 영역에서 10문항 중 4문항을 맞춰 40%의 정답률을 기록했습니다. 기본 문법은 부분적으로 이해하고 있으나, 가정법·도치·시제 등 핵심 구조에서 취약점이 드러납니다.

Q2
정답: D / 제출: A
분사구문에서 주절 주어와 분사 주어가 다를 때 독립분사구문을 써야 합니다. ④번 forecasted는 forecast가 올바른 과거분사 형태입니다.
Q4
정답: A / 제출: B
insist that + 주어 + 동사원형 (가정법 현재)을 사용해야 합니다. attends가 아니라 attend가 맞습니다.
Q6
정답: C / 제출: A
도치 구문에서 Not until이 문두에 오면 주어-동사 도치가 필요합니다. did the audience discover가 정답입니다.
Q8
정답: A / 제출: C
the more~, the more~ 비교급 구문입니다. the more adept가 정답이며, 비교급 강조 구문의 기본 패턴을 익혀야 합니다.
외 2문항 오답 (상세 채점표 참조)
어법 영역에서 4문항 정답 처리되었습니다.

학습 포인트

  • 가정법 현재 (insist/recommend/imperative that + 원형) 패턴 집중 학습
  • 도치 구문 유형별 정리 (부정 부사, only, not until 등)
  • 분사구문의 능동(~ing)/수동(p.p.) 구별 연습
  • the 비교급, the 비교급 구문 반복 훈련

어휘 영역 분석

04 어휘 분석 (6/10, 60%)

영역 개요

어휘 영역에서 10문항 중 6문항을 맞춰 60%를 기록했습니다. 기본 어휘력은 갖추고 있으나 고난도 유의어 구별과 문맥 적합 어휘 판단에서 실수가 있습니다.

Q13
정답: C / 제출: A
adroit는 '능숙한(skillful)'이라는 뜻입니다. clumsy(서투른)와 반의어 관계이며, dexterous와 유의어입니다.
Q17
정답: B / 제출: A
resolute는 '단호한, 굳건한'이라는 뜻입니다. determined, unwavering과 유의어이며, 문맥상 setbacks에도 불구하고 끝까지 해낸 것을 나타냅니다.
Q18
정답: A / 제출: C
poignant는 '가슴 아픈, 애절한'이라는 뜻입니다. 사랑과 상실의 이야기라는 문맥에서 touching, moving과 유사합니다.
Q20
정답: A / 제출: B
substantive는 '실질적인, 본질적인'이라는 뜻입니다. 빈 수사학과 대비되는 맥락에서 meaningful, significant와 유사합니다.
어휘 영역에서 6문항 정답 처리되었습니다.

학습 포인트

  • 편입 고빈출 형용사 300개 집중 암기 (유의어 그룹별)
  • 문맥 단서를 활용한 어휘 추론 연습
  • 반의어 짝 학습으로 의미 변별력 강화
  • 매일 어휘 테스트 20개 (누적 복습)

장문독해 영역 분석

05 장문독해 분석 (12/20, 60%)

영역 개요

장문독해 영역에서 20문항 중 11문항을 맞춰 55%를 기록했습니다. 쉬운 주제 파악은 가능하나, 추론·세부 정보·어휘 유추 문제에서 오답이 집중됩니다.

Q23
정답: C / 제출: A
스토아 철학의 핵심 목표를 묻는 문제입니다. 'inner peace and resilience'가 정답이며, 지문의 'apatheia(평정)'와 연결됩니다.
Q27
정답: C / 제출: A
긱 이코노미에서 근로자 혜택을 묻는 문제입니다. 'Greater autonomy over work schedule'이 정답이며, 지문의 'set their own hours'와 직결됩니다.
Q29
정답: C / 제출: A
긱 이코노미의 핵심 쟁점을 추론하는 문제입니다. 유연성과 장기 안정성 간의 갈등이 지문 전체의 구조적 대립입니다.
Q33
정답: B / 제출: A
paramount의 이유를 묻는 문제입니다. 우주의 운명을 이해하는 데 결정적이라는 지문의 마지막 문장이 근거입니다.
외 3문항 오답 (상세 채점표 참조)
장문독해 영역에서 12문항 정답 처리되었습니다.

학습 포인트

  • 지문의 핵심 논지(thesis) 파악 훈련 — 서론/결론 집중 읽기
  • 추론 문제 유형 연습: 지문 근거를 반드시 찾는 습관
  • 선택지 소거법 훈련: 오답의 함정 패턴 분류
  • 매일 2지문씩 시간 제한 풀이 + 오답 분석

종합 솔루션 및 학습 전략

06 맞춤형 학습 솔루션

홍길동 학생은 전체 40문항 중 22문항을 맞춰 55%의 정답률을 보였습니다. 전형적인 중위권 성적으로, 어법이 가장 취약하고 어휘·독해는 기본기는 있으나 불안정합니다. 전 영역 균형 보강이 필요합니다.

PRIORITY 1

어법 체계적 보강 (최우선)

  • 가정법·도치·분사구문 3대 취약 유형 집중 훈련
  • 매일 어법 문제 10문항 + 오답 노트 필수 작성
  • 문법 규칙을 예문 3개씩 암기하여 체화
PRIORITY 2

독해 정확도 향상

  • 추론/세부 정보 문제 유형별 집중 풀이
  • 지문 구조 분석 훈련: 주제문·근거문 표시하며 읽기
  • 매일 편입 기출 독해 2지문씩 정독 + 오답 분석
PRIORITY 3

어휘력 확장

  • 편입 고빈출 형용사/동사 300개 집중 암기
  • 유의어·반의어 그룹 학습으로 변별력 강화
  • 어원(접두사/접미사) 기반 파생어 학습

길동 학생은 기본기를 갖추고 있어 체계적 보강만으로 큰 폭의 성적 향상이 가능합니다. 어법 패턴 학습과 독해 정확도 훈련에 집중한다면, 목표 대학 합격은 충분히 달성 가능합니다. 화이팅!
