# 🎨 Gradient Color Documentation

## 1️⃣ **Pastel Coral Cream Gradient**

* **HEX:** `#EEF3D2 → #FC8884`
* **분위기:** 따뜻하고 부드러운 코랄빛
* **특징 설명:**

  * 상단은 크림 톤의 연노랑 (#EEF3D2)
  * 하단은 생기 있는 코랄핑크 (#FC8884)
  * 감성적이며 부드러운 인상, 따뜻한 온도감
* **추천 활용:** 봄 시즌 교재 표지, 감성적인 브랜드 디자인, 따뜻한 톤의 UI

---

## 2️⃣ **Deep Blue to Mint Gradient**

* **HEX:** `#014871 → #D7EDE2`
* **분위기:** 청량하고 차분한 블루톤
* **특징 설명:**

  * 딥블루(#014871)에서 민트톤(#D7EDE2)으로 부드럽게 이동
  * 차분하면서도 세련된 그라데이션
  * 신뢰감과 프로페셔널한 인상
* **추천 활용:** 교육/테크 브랜드, 대시보드 배경, 시원한 테마의 인쇄물

---

## 3️⃣ **Lavender Mint Pastel Gradient**

* **HEX:** `#EAD6EE → #A0F1EA`
* **분위기:** 몽환적이고 청량한 파스텔
* **특징 설명:**

  * 라벤더 퍼플에서 민트 시안으로의 부드러운 전환
  * 여성적이면서 미래지향적인 이미지
  * 눈의 피로도가 적고 고급스러운 느낌
* **추천 활용:** AI·테크 교재 표지, 웹/앱 UI 배경, 감성형 디자인
