258번 — 문장 기능 파악

핵심 개념: 문장 기능 유형

지문 내 문장은 주제 제시, 예시/근거, 부연 설명, 반론/양보, 결론/요약 중 하나의 기능을 수행합니다.

For instance, However, In other words 등의 담화 표지(discourse marker)를 활용해 기능을 판별합니다.

해석: "푸코가 도입한 '통치성' 개념은 근대 권력의 본질에 대해 독특한 관점을 제시한다. 이것은 18세기에 등장한 특정한 '통치 기술'을 기술하며, 영토 지배에서 '인구' 관리로 초점이 이동했음을 보여준다."

핵심 키워드

구분표현 및 키워드
핵심 키워드governmentality, biopower, art of government
정답 방향 (Positive)specific example, illustrating, For instance
반대 방향 (Negative)counterargument, refutation, contrast
구조 분석

방향: POSITIVE (개념 정의 → 구체화)

구조: 개념 도입(1-2문장) → 세부 설명(3-4문장) → 예시 제시(5문장) → 결론(6-7문장)

5번째 문장의 "For instance"가 예시 기능임을 명확히 표시합니다.

정답: ③ To provide a specific example illustrating the tactics of governmentality.
오답 ① 분석

"To introduce a counterargument" — 지문에 반론이나 양보 표현이 없습니다. however, but, on the other hand 등이 부재합니다.

오답 ② 분석

"To state the main thesis" — 주제문은 첫 문장에 이미 제시되었습니다. 5번째 문장은 보조적 역할(예시)입니다.

오답 ④ 분석

"To offer a concluding summary" — 결론은 마지막 문장("Thus, governmentality reveals...")이 담당합니다. 5번째 문장은 중간에 위치해 결론 기능을 하지 않습니다.

학생 답 분석

왜 이 답을 골랐을까: "예시"와 "부연 설명"을 혼동. "For instance"를 단순한 추가 설명으로 해석한 것으로 보임

오류 패턴: 담화 표지 간과 — 핵심 시그널 단어를 놓치고 문맥만으로 판단

집중 포인트: For instance, For example = 예시 / In other words, That is = 부연 / However, But = 반전

보충 문법 보충 설명
담화 표지 (Discourse Markers) 총정리

예시 표지: For instance, For example, such as, including, namely

부연 표지: In other words, That is, i.e., To put it differently

대조 표지: However, But, On the other hand, In contrast, Nevertheless

결론 표지: Thus, Therefore, Hence, Consequently, In conclusion

개념 보충 개념 보충 학습

1. 문장 기능 파악 전략

핵심 개념

영어 지문에서 각 문장은 고유한 기능(function)을 갖습니다. 기능은 크게 5가지: 주제 제시, 예시/근거, 부연 설명, 반론/양보, 결론/요약.

문장 앞의 담화 표지(discourse marker)가 가장 강력한 단서입니다.

핵심 예시: 1. "For instance, the collection of data allows..." → 예시 기능 2. "In other words, reality is unthinkable." → 부연/재진술 기능 3. "However, this approach has limitations." → 반론/전환 기능

예문 연습

1. The study concluded that expectations affect performance. → 결론 기능 2. For example, teachers provided more challenging material. → 예시 기능 3. Nevertheless, the methodology has been questioned. → 양보/반전 기능
헷갈리는 포인트

"예시"와 "부연 설명"의 차이:

예시(For instance) = 구체적 사례 제시 / 부연(In other words) = 같은 내용을 다른 표현으로 반복

예시는 새로운 정보를 추가하고, 부연은 기존 정보를 재표현합니다.

연습 1 문장 기능 파악 전략 연습문제 (5문제)

1. The sentence "For example, demographic data collection enables state intervention" functions primarily to:

① state the main thesis   ② provide a counterargument   ③ offer a specific example   ④ summarize the passage   ⑤ define a key term

2. "In other words, reality independent of the human mind is unthinkable." This sentence serves to:

① introduce a new concept   ② rephrase the preceding idea   ③ present a counterexample   ④ conclude the argument   ⑤ raise a question

3. "However, the affective fallacy warns against evaluating literature by emotional response." This sentence:

① supports the previous claim   ② provides an example   ③ introduces a contrasting view   ④ defines a technical term   ⑤ summarizes the passage

4. "Thus, governmentality reveals how deeply political power is invested in biological existence." This sentence:

① presents a new argument   ② offers an illustration   ③ restates the introduction   ④ draws a conclusion   ⑤ raises a counterpoint

5. "The students labeled as bloomers showed significantly greater improvement." This sentence:

① describes the methodology   ② states the hypothesis   ③ presents the key finding   ④ offers a critique   ⑤ introduces the topic
