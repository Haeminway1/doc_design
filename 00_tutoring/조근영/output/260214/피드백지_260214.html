학습 요약

아쉬운 점: 2형식 동사 뒤에 부사(-ly)를 넣으려는 실수 반복, hard/hardly 혼동쌍 구분 미숙

취약 영역문제 수심각도핵심
A. 2형식 동사 + 형용사 보어 3문제 최우선 seem, remain, prove 뒤에 형용사 보어가 와야 하는데 부사(-ly)를 선택
B. 동사 수식 부사 2문제 높음 동작동사 뒤에는 부사가 와야 하는데 형용사를 선택
C. 부사 + 형용사 수식 2문제 높음 형용사 앞에서 정도를 나타내는 부사 자리에 형용사를 선택
D. hard/hardly, high/highly 혼동쌍 1문제 높음 -ly를 붙이면 의미가 완전히 달라지는 부사 혼동
E. 문장 부사 (보충) 1문제 보통 문두에서 문장 전체를 수식하는 부사 자리에 형용사를 선택
학습 우선순위: 테마 A (2형식 보어) → 테마 B (동사 수식 부사) → 테마 C (부사+형용사) → 테마 D (혼동쌍)

Section 2 해설 파트

테마 A: 2형식 동사 + 형용사 보어 — #1, #4, #5

핵심 개념 2형식 동사는 "주어 = 보어" 관계를 만든다. 보어 자리에는 반드시 형용사가 온다.

2형식 동사 목록 (반드시 암기!)

그룹동사의미
상태 유지be, remain, stay, keep~이다, ~인 채로 있다
상태 변화become, get, grow, turn, go, come~해지다
외관/인식seem, appear, look, prove~인 것 같다, ~로 판명되다
감각feel, taste, smell, sound~하게 느끼다/맛나다/냄새나다/들리다

핵심 판별법: "주어 = 보어" 테스트

The solution seems simple. → The solution = simple (O) → 형용사! She remained calm. → She = calm (O) → 형용사! The project seemed impossible. → The project = impossible (O) → 형용사!

왜 틀렸을까?

문제함정올바른 판단
#1 seems simple/simplyseem 뒤에 -ly를 붙이려는 유혹seem = 2형식 → 보어는 형용사
#4 seemed impossible/impossibly같은 유혹seem = 2형식 → 보어는 형용사
#5 remained calm/calmlyremain 뒤에 -ly를 붙이려는 유혹remain = 2형식 → 보어는 형용사

특별 주의: 같은 동사의 2형식 vs 동작동사

동사2형식 (형용사 보어)동작동사 (부사)
lookShe looked nervous. (~해 보였다)She looked nervously at the door. (긴장하며 쳐다보았다)
feelI felt sad. (슬펐다)I felt carefully for the switch. (조심히 더듬었다)
growThe tree grew tall. (커졌다)The company grew steadily. (꾸준히 성장했다)
판별 팁 동사 뒤 빈칸을 볼 때 → "주어 = X"가 성립하면 형용사, 안 되면 부사!

테마 B: 동사 수식 부사 — #3, #5

핵심 개념 동작을 나타내는 일반동사 뒤에는 부사가 온다. "어떻게?"에 대한 답이다.
공식: 주어 + 동사 + 부사 (+ 목적어)

왜 틀렸을까?

문제함정올바른 판단
#3 drove carefullycareful을 넣으려는 유혹drove = "운전하다" 동작동사 → 부사
#5 handled professionallyprofessional을 넣으려는 유혹handled = "처리하다" 동작동사 → 부사

판별법: "주어 = X" 테스트

She drove carefully. → She = carefully? (X) → 부사! (그녀가 careful한 게 아니라, 운전을 carefully 한 것) She handled it professionally. → She = professionally? (X) → 부사!
핵심: 한 문장에 2형식 + 동작동사가 같이 나올 때 (#5) She handled the crisis professionally ← 동작동사 → 부사
and remained calm under pressure. ← 2형식 동사 → 형용사

한 문장에서 동사마다 따로 판단해야 한다!

테마 C: 부사 + 형용사 수식 — #1, #3

핵심 개념 형용사 앞에서 정도를 나타내는 것은 부사이다.
공식: 부사 + 형용사

대표 예시

부사 + 형용사의미
extremely challenging극도로 도전적인
dangerously slippery위험할 정도로 미끄러운
highly appreciated매우 높이 평가받는
incredibly difficult믿을 수 없을 정도로 어려운

왜 틀렸을까?

문제함정올바른 판단
#1 extremely challengingextreme을 넣으려는 유혹extreme(형용사) + challenging(형용사) = 형용사끼리 나란히 불가
#3 dangerously slipperydangerous를 넣으려는 유혹dangerous(형용사) + slippery(형용사) = 비문
판별 팁 "_____ + 형용사" 구조가 보이면 → 빈칸은 99% 부사!
_____ challenging → extremely (부사) / _____ slippery → dangerously (부사)

테마 D: hard/hardly, high/highly 혼동쌍 — #2

핵심 개념 일부 부사는 -ly 없이도 부사로 쓰이며, -ly를 붙이면 뜻이 완전히 달라진다!
부사 (원형)의미-ly 부사의미
hard열심히, 세게hardly거의 ~않다
high높이 (물리적)highly매우, 대단히 (추상적)
late늦게lately최근에
near가까이nearly거의
close가까이closely면밀히, 밀접하게
free무료로freely자유롭게

#2 해설

work hard = 열심히 일하다 (O) work hardly = 거의 일하지 않다 (의미가 완전 반대!) highly appreciated = 매우 높이 평가받는 (추상적 "높이") high appreciated = (X) high는 물리적 높이에만 사용

추가 예문

원형 부사-ly 부사
The bird flew high in the sky. (물리적 높이)He is highly respected. (추상적)
He came home late. (늦게)I haven't seen him lately. (최근에)
Don't go too near the fire. (가까이)It's nearly impossible. (거의)
주의 hard/hardly, high/highly는 편입 시험 단골 출제! 반드시 의미 차이를 외워두세요.

보충: 문장 부사 — #4

핵심 개념 문장 맨 앞에서 문장 전체를 수식하는 부사. 화자의 태도/판단을 나타낸다.
공식: -ly 부사, + 완전한 문장

대표 문장 부사

부사의미
Unfortunately불행히도
Initially처음에는
Generally (speaking)일반적으로
Surprisingly놀랍게도
Personally개인적으로
Honestly솔직히
판별법 문두 + 콤마(,) + 완전한 문장 → 문장 부사 → -ly 형태
___, the project seemed impossible. → Initially (부사)

지각동사 (see, hear, feel)
