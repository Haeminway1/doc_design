make의 두 얼굴 — 4형식 vs 5형식

4형식: make + I.O + D.O ("만들어 주다") I will make you some coffee. → you(I.O) ≠ some coffee(D.O) → 4형식! 5형식: make + O + O.C ("~하게 만들다") The news made him happy. → him(O) = happy(O.C) → 5형식!
판별법 I.O ≠ D.O → 4형식 (you ≠ coffee)
O = O.C → 5형식 (him = happy)

F 복습 체크리스트
1형식 (완전자동사)
happen, live, go on, go up, go off, break out → 자동사 → 1형식
부사(unexpectedly, alone, suddenly)는 보어가 아니다 → 수식어
2형식 (연결동사)
remain, stay, come, seem, appear, prove, turn out + 형용사(보어) → 2형식
"S = C" 관계 확인: situation = unchanged, dream = true, argument = logical
같은 동사: came home(1형식) vs came true(2형식) — 뒤에 형용사 있으면 2형식
3형식 (타동사 + O)
관사 + 형용사 + 명사 = 하나의 O 덩어리
구동사 타동사: look for, take care of, deal with → V + O = 3형식
provide/supply A with B = 3형식 (전치사 있으면 3형식!)
4형식 (V + I.O + D.O)
make/pay/promise/get + I.O(사람) + D.O(사물) → 4형식
I.O ≠ D.O → 4형식 (you ≠ coffee)
3형식 전환: to(give, tell, pay, promise) / for(make, buy, get, cook)
5형식 (V + O + O.C)
make/keep/leave/find/believe/consider/elect + O + O.C → 5형식
O = O.C 관계 확인: her = speechless, city = safe, lecture = boring
believe/consider + O + to be + C → 5형식 (to부정사 보어)
4형식 vs 5형식 핵심 구분
I.O ≠ D.O → 4형식 (make you coffee: you ≠ coffee)
O = O.C → 5형식 (make him happy: him = happy)
