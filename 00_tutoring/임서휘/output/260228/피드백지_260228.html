Section 2 해설 파트

13번 — 의미 혼동

immune adj. → 한국어 뜻 쓰기

immune = 면역의, ~에 영향을 받지 않는

im-(not) + mune(의무) → 의무에서 벗어난 → 면역의

• immune to disease = 질병에 면역인

• immune system = 면역 체계

⚠ 면하다 vs 면역의

면하다 = to avoid, to escape (위기를 면하다)

면역의(immune) = 외부 자극에 영향을 받지 않는 상태

'면'이라는 글자는 같지만 의미가 완전히 다릅니다.

학생 답 분석

어디서 막혔을까: 'immune'의 '면(免)'과 '면하다'의 '면'을 연결하여 추측한 것으로 보임

오류 패턴: 어근 일부만 기억하고 전체 의미를 추측

집중 포인트: immune = '면역의'로 통째로 암기. immunity(면역), immunize(면역시키다) 함께 외우기

20번 — 의미 혼동

conscious adj. → 한국어 뜻 쓰기

conscious = 의식하는, 자각하는

con-(함께) + sci(알다) → 함께 아는 → 의식하는

• be conscious of = ~을 의식하다

• a conscious decision = 의식적인 결정

관련 어휘:

• conscious (adj) 의식하는 ↔ unconscious 무의식의

• consciousness (n) 의식

• conscience (n) 양심 ← 철자 주의!

학생 답 분석

어디서 막혔을까: '의식'과 관련된 단어임을 어렴풋이 기억했으나 정확한 뜻을 쓰지 못함

오류 패턴: 어근의 느낌만 기억하고 정확한 한국어 대응어를 모름

집중 포인트: conscious = '의식하는' + be conscious of 구문 함께 암기

27번 — 유사 단어 혼동

election n. → 한국어 뜻 쓰기

election = 선거

e-(밖으로) + lect(고르다) → 골라내다 → 선거

• presidential election = 대통령 선거

• elect a leader = 지도자를 선출하다

⚠ 선거 vs 선언

단어뜻영어
선거(選擧)대표를 뽑는 행위election
선언(宣言)공식적으로 알림declaration

'선'자가 같지만 전혀 다른 단어입니다.

학생 답 분석

어디서 막혔을까: '선(選/宣)'이라는 한자 음이 같아 선거와 선언을 혼동

오류 패턴: 한자어 음독 유사성에 의한 혼동

집중 포인트: election = elect(선출하다) + -ion → 선거. declaration = declare(선언하다) + -tion → 선언

32번 — 유사 단어 혼동

pollution n. → 한국어 뜻 쓰기

pollution = 오염

pollut(더럽히다) + -ion → 더럽힘 → 오염

• air pollution = 대기 오염

• water pollution = 수질 오염

• pollute (v) 오염시키다 / pollutant (n) 오염 물질

⚠ 오염 vs 확산

오염(pollution) = 유해 물질로 더러워지는 것

확산(spread/diffusion) = 퍼져 나가는 것

오염이 '확산'될 수는 있지만, pollution 자체의 뜻은 '오염'입니다.

학생 답 분석

어디서 막혔을까: 오염의 '확산'이라는 연관 개념을 떠올렸으나 pollution 자체의 뜻과 혼동

오류 패턴: 단어의 뜻 대신 연관 개념을 답으로 작성

집중 포인트: pollution = 오염 (pollute → 오염시키다). spread/diffusion = 확산

33번 — 유사 단어 혼동

profession n. → 한국어 뜻 쓰기

profession = 직업, 전문직

pro-(앞에서) + fess(말하다) → 공개적으로 선언하다 → (전문 자격을 선언한) 직업

• medical profession = 의료 전문직

• professional (adj/n) = 전문적인 / 전문가

⚠ profession vs professor

단어뜻품사
profession직업, 전문직명사
professor교수명사

어근은 같지만(profess-) 접미사가 다릅니다: -ion(행위/상태) vs -or(사람)

학생 답 분석

어디서 막혔을까: profession과 professor의 철자가 유사하여 '교수'로 답함

오류 패턴: 파생어 간 혼동 — 접미사 구분 미숙

집중 포인트: profession(-ion=것) = 직업, professor(-or=사람) = 교수. 접미사로 구분!

48번 — 의미 혼동

excessive adj. → 한국어 뜻 쓰기

excessive = 과도한, 지나친

ex-(넘어서) + cess(가다) → 한계를 넘어간 → 과도한

• excessive spending = 과도한 지출

• excessive heat = 지나친 더위

• exceed (v) 초과하다 / excess (n) 초과, 과잉

⚠ excessive ≠ 제어하는

제어하는 = controlling (control의 파생)

과도한 = excessive (exceed의 파생)

정반대 방향의 의미입니다.

학생 답 분석

어디서 막혔을까: 단어의 의미를 기억하지 못하고 ex- 접두사에서 다른 단어를 추측한 것으로 보임

오류 패턴: 모르는 단어에 대한 추측 — 관련 없는 의미 작성

집중 포인트: excessive = exceed(초과하다)의 형용사형 = 과도한. ex-(밖으로, 넘어서) 기억

54번 — 철자 오류

번영하는 adj. → 영어 철자 쓰기

prosperous [프라스퍼러스]

prosper(번영하다) + -ous(형용사)

철자 분해: pro-sper-ous

• a prosperous nation = 번영하는 국가

• prosper (v) 번영하다 / prosperity (n) 번영

✅ 올바른 철자

p-r-o-s-p-e-r-o-u-s

학생 작성: persporoue → 자음·모음 순서 뒤바뀜

학생 답 분석

어디서 막혔을까: prosper 어근을 알고 있으나 철자 순서를 정확히 기억하지 못함

오류 패턴: 음절 순서 전도 (pro→per, sper→spor)

집중 포인트: prosper + ous = prosperous. 동사 prosper를 먼저 확실히 외운 뒤 -ous 붙이기

57번 — 품사 혼동 (형용사→부사)

흔히 adv. → 영어 철자 쓰기

commonly = 흔히, 일반적으로

common(흔한, adj) + -ly = commonly(흔히, adv)

• It is commonly believed that... = 흔히 ~라고 믿어진다

⚠ 형용사 vs 부사

문제에서 '흔히 adv.'라고 품사를 명시했습니다.

common = 흔한 (형용사) ← 이것을 적으면 오답!

commonly = 흔히 (부사) ← 정답

학생 답 분석

어디서 막혔을까: common의 뜻은 알고 있으나 부사형 -ly를 붙이는 것을 간과

오류 패턴: 품사 변환 규칙 미적용 — adv. 표시를 놓침

집중 포인트: 문제의 품사 표시(adj./adv./n./v.)를 반드시 확인! adv. → -ly 붙이기

58번 — 품사 혼동 (형용사→부사)

효과적으로 adv. → 영어 철자 쓰기

effectively = 효과적으로

effective(효과적인, adj) + -ly = effectively(효과적으로, adv)

• communicate effectively = 효과적으로 소통하다

⚠ 같은 패턴 반복!

Q57 common→commonly와 동일한 실수입니다.

한국어 '~으로' '~하게' '~히' = 영어 부사 = 형용사 + -ly

학생 답 분석

어디서 막혔을까: effective는 알지만 -ly를 붙여야 한다는 인식이 부족

오류 패턴: 부사 형성 규칙 미적용 (Q57과 동일 패턴)

집중 포인트: '~으로/~하게/~히' = adverb = 반드시 -ly 확인!

68번 — 철자 오류 (-lly 규칙)

성공적으로 adv. → 영어 철자 쓰기

successfully = 성공적으로

successful + -ly = successfully

⚡ -ful + -ly = -fully (l이 두 개!)

• successfully complete = 성공적으로 완료하다

✅ 철자 규칙

success → successful → successfully

-ful로 끝나는 형용사 + ly → -fully (ll 유지)

학생: successfuly (l 1개) → successfully (l 2개)

학생 답 분석

어디서 막혔을까: -ful + -ly 조합에서 l을 하나 빠뜨림

오류 패턴: 이중 자음 규칙 미숙 (-ful + -ly = -fully)

집중 포인트: -ful + -ly = -fully: beautifully, carefully, successfully, hopefully 패턴 암기

69번 — 철자 오류 (-lly 규칙)

일반적으로 adv. → 영어 철자 쓰기

normally = 일반적으로, 보통

normal + -ly = normally

⚡ -al + -ly = -ally (l이 두 개!)

• I normally wake up at 7. = 나는 보통 7시에 일어난다.

✅ 철자 규칙

norm → normal → normally

-al로 끝나는 형용사 + ly → -ally (ll 유지)

학생: normaly (l 1개) → normally (l 2개)

학생 답 분석

어디서 막혔을까: -al + -ly 조합에서 l을 하나 빠뜨림 (Q68과 동일 패턴)

오류 패턴: 이중 자음 규칙 미숙 — -al + -ly = -ally

집중 포인트: -al + -ly = -ally: normally, finally, occasionally, accidentally 패턴 암기

75번 — 철자 오류 (심각)

희생 n./v. → 영어 철자 쓰기

sacrifice [새크리파이스]

sacri-(신성한) + fice(만들다) → 신에게 바치다 → 희생

철자 분해: sac-ri-fice

• make a sacrifice = 희생하다

• sacrifice one's time = 시간을 희생하다

✅ 올바른 철자

s-a-c-r-i-f-i-c-e (9글자)

학생 작성: softcity → 완전히 다른 단어

발음 [새크리파이스]를 기억하며 철자 연습 필요

학생 답 분석

어디서 막혔을까: sacrifice의 철자를 전혀 기억하지 못하고 음감으로 추측

오류 패턴: 단어 철자 미암기 — 소리 기반 추측 실패

집중 포인트: sacrifice = sacri(신성한) + fice(만들다). 발음과 어원 함께 반복 암기

80번 — 철자 오류

순서 n. → 영어 철자 쓰기

sequence [시퀀스]

sequ-(따르다) + -ence(명사)

철자 분해: se-quence

• in sequence = 순서대로

• a sequence of events = 일련의 사건들

✅ 철자 교정

학생: sequance → 정답: sequence

-ence 접미사: sequence, difference, reference

학생 답 분석

어디서 막혔을까: -ence와 -ance 접미사를 혼동하여 'a'로 작성

오류 패턴: -ence vs -ance 접미사 혼동

집중 포인트: sequence = -ence 그룹. -ance: distance, importance / -ence: sequence, difference

81번 — 철자 오류

종 n. → 영어 철자 쓰기

species [스피시즈]

speci-(보다) + -es → 겉모습으로 분류한 것 → 종(種)

⚡ 단수·복수 동형: a species / two species

• endangered species = 멸종 위기종

✅ 철자 교정

학생: specis → 정답: species

-ies 끝: species (i 다음에 e가 하나 더!)

학생 답 분석

어디서 막혔을까: species의 -ies 부분에서 e를 빠뜨림

오류 패턴: 모음 연속(ie) 누락

집중 포인트: species = speci + es. 단복수 동형(a species / many species) 함께 기억

85번 — 유사 단어 혼동

사건 n. → 영어 철자 쓰기

사건 = incident / event

incident = (불쾌한) 사건, 사고 → in-(안으로) + cid(떨어지다)

event = (중요한) 사건, 행사 → e-(밖으로) + vent(오다)

⚠ incident vs accident

영어한국어뉘앙스
incident사건일반적인 사건·일
accident사고뜻하지 않은 사고 (교통사고 등)

사건(incident) ≠ 사고(accident)

학생 답 분석

어디서 막혔을까: '사건'과 '사고'를 혼동하여 accident(사고)를 작성

오류 패턴: 한국어 유사어(사건/사고) → 영어 대응어 혼동

집중 포인트: 사건=incident, 사고=accident. 한국어 뜻을 정확히 구분한 뒤 영어 대응

91번 — 철자 오류

분석하다 v. → 영어 철자 쓰기

analyze [애널라이즈]

ana-(위로) + lyze(풀다) → 분해하여 살펴보다 → 분석하다

철자 분해: a-na-lyze

• analyze data = 데이터를 분석하다

• analysis (n) 분석 / analyst (n) 분석가

✅ 철자 교정

학생: analizoe → 정답: analyze

핵심: -lyze (y + z + e)

학생 답 분석

어디서 막혔을까: analyze의 대략적인 발음은 아나 모음·자음 배열을 정확히 기억하지 못함

오류 패턴: 모음 위치 혼동 (i↔y, 불필요한 o 추가)

집중 포인트: analyze = ana + lyze. analysis(분석)과 함께 외우면 lyze/lysis 패턴 기억에 도움

95번 — 품사 혼동 (형용사↔부사)

자동의 adj. → 영어 철자 쓰기

automatic = 자동의 (형용사)

auto-(스스로) + matic(움직이는)

• automatic door = 자동문

• automatic update = 자동 업데이트

⚠ automatic vs automatically

문제: '자동의 adj.' → 형용사를 요구

automatic (adj) = 자동의 ← 정답

automatically (adv) = 자동으로 ← 부사 (오답)

학생은 부사형을 적었고, 철자도 부정확(matically → matically)

학생 답 분석

어디서 막혔을까: automatic과 automatically를 구분하지 못하고 더 긴 형태를 작성

오류 패턴: 품사 표시(adj.) 미확인 + 부사형과 형용사형 혼동

집중 포인트: adj. = 형용사 요구 → automatic. adv. = 부사 요구 → automatically. 품사 표시 꼭 확인!

1. 부사 형성 규칙 (-ly 패턴 완전 정복)

형용사 → 부사 변환 규칙

영어에서 부사는 대부분 형용사 + -ly로 만듭니다.

규칙형용사부사
기본: +lycommoncommonly
-al +ly = -allynormalnormally (ll!)
-ful +ly = -fullysuccessfulsuccessfully (ll!)
-le → -lysimplesimply
-y → -ilyeasyeasily
-ic → -icallyautomaticautomatically

⚠ 주의! l이 두 개 되는 경우

-al + ly = -ally (normally, finally)

-ful + ly = -fully (successfully, carefully)

common → commonly (기본 +ly) normal → normally (-al +ly = -ally) successful → successfully (-ful +ly = -fully) simple → simply (-le → -ly) easy → easily (-y → -ily) automatic → automatically (-ic → -ically)

시험에서 자주 틀리는 포인트

1. 문제의 품사 표시(adj./adv.)를 반드시 확인하세요!

2. -al/-ful로 끝나는 형용사 → 부사로 바꿀 때 l이 두 개가 됩니다.

3. 형용사 자체를 쓰면 오답입니다 (common ≠ commonly).

2. 혼동하기 쉬운 유사 단어 구분

형태가 비슷하지만 뜻이 다른 단어들

영어에는 철자나 발음이 비슷해서 헷갈리기 쉬운 단어 쌍이 많습니다. 정확한 의미 차이를 알아야 시험에서 실수를 방지할 수 있습니다.

혼동 쌍단어 A단어 B
선거/선언election(선거)declaration(선언)
오염/확산pollution(오염)diffusion(확산)
직업/교수profession(직업)professor(교수)
사건/사고incident(사건)accident(사고)
election(선거) ≠ declaration(선언) — elect=선출하다, declare=선언하다 pollution(오염) ≠ diffusion(확산) — pollute=더럽히다, diffuse=퍼뜨리다 profession(직업) ≠ professor(교수) — -ion=것, -or=사람 incident(사건) ≠ accident(사고) — 일반 사건 vs 뜻밖의 사고

구분 전략

1. 접미사로 구분: -ion(상태/행위) vs -or/-er(사람)

2. 어근으로 구분: elect(고르다)→선거 vs declare(알리다)→선언

3. 한국어 뜻을 정확히: 사건≠사고, 오염≠확산
