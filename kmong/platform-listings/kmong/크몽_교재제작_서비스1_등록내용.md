# 크몽 서비스 등록 — 학원/교육 교재·문제집 PDF 맞춤 제작

> 카테고리: 디자인 > 인쇄물 디자인 (또는 IT·프로그래밍 > 전자책 제작)
> 핵심: 전문 디자인 시스템 + 빌드 파이프라인으로 고품질 교재 제작

---

## 1. 서비스 제목

학원 강사 출신이 만드는 교재·문제집 PDF 맞춤 디자인 (A4 인쇄용)

---

## 2. 서비스 설명

안녕하세요, 학원 강사 출신 교재 전문 디자이너 해민입니다.

5년 넘게 영어학원에서 직접 강사로 일하면서
수업에 필요한 교재, 문제집, 유인물, 워크시트를
직접 만들어 사용해왔습니다.

현재 전문 교재 디자인 시스템을 보유하고 있으며,
20종 이상의 디자인 컴포넌트와 8종 시각 테마를 활용해
깔끔하고 전문적인 교재를 빠르게 제작합니다.

"교재 내용은 있는데 디자인이 안 되는" 선생님,
"전문적인 교재를 학원 브랜딩에 맞춰 만들고 싶은" 원장님,
"시험 문제집을 깔끔하게 편집하고 싶은" 강사님께 딱 맞는 서비스입니다.

A4 인쇄에 최적화된 PDF로 납품합니다.

---

## 3. 주요특징

1. 학원 강사 출신 — 교재 구성/레이아웃을 교육자 관점에서 설계
2. 전문 디자인 시스템 보유 — 20종 컴포넌트 + 8종 테마로 일관된 품질
3. A4 인쇄 최적화 — PDF 출력 시 깨짐/잘림 없는 완벽한 규격
4. 빠른 제작 — 자체 빌드 파이프라인으로 대량 페이지도 신속 제작
5. 다양한 교재 유형 — 문법/독해/어휘/문제집/워크시트/유인물 모두 가능

---

## 4. 제작 가능 교재 유형

### 문제집/시험지
- 객관식/주관식/서술형 문제 편집
- 정답 해설지 별도 제작
- OMR 답안지 포함 가능

### 교재/워크북
- 개념 설명 + 예제 + 연습문제 구성
- 컨셉 박스, 요약 박스, 주의사항 박스 등 구조화된 디자인
- 단원별 목차 + 페이지 번호

### 유인물/워크시트
- 수업용 1~2페이지 유인물
- 빈칸 채우기, 연결하기 등 활동지
- 일일 테스트/퀴즈지

### 어휘집/단어장
- Day별 구성 (단어 + 뜻 + 예문)
- 테스트 페이지 자동 생성
- A4 또는 핸드북 사이즈

### 커리큘럼/안내문
- 학원 커리큘럼 브로셔
- 학부모 안내문
- 수업 계획서

## 6. 서비스 제공절차

### STEP 1. 상담 (당일)
- 교재 유형, 과목, 분량 파악
- 원고/콘텐츠 확인 (텍스트, 이미지 등)
- 디자인 방향 협의 (테마, 컬러, 분위기)

### STEP 2. 디자인 시안 (1~2일)
- 표지 + 본문 2~3페이지 시안 제작
- 의뢰인 피드백 반영

### STEP 3. 전체 제작 (2~7일)
- 전체 페이지 디자인 + 편집
- 표, 이미지, 아이콘 배치
- 인쇄 최적화 작업

### STEP 4. 검수 + 납품 (1일)
- 최종 PDF 검수 (오탈자, 레이아웃, 인쇄 테스트)
- 수정 사항 반영
- 인쇄용 PDF + 원본 파일 납품

## 9. 의뢰인 준비사항

1. **원고/콘텐츠**: 텍스트 파일 (한글, 워드, 구글독스, 텍스트 어떤 형태든 가능)
2. **이미지** (있으면): 교재에 들어갈 사진, 그림, 도표
3. **참고 자료** (있으면): "이런 느낌으로 만들어주세요" — 기존 교재, 디자인 레퍼런스
4. **학원 로고** (있으면): PNG/SVG 형태
5. **선호 색상/테마** (없으면 추천해드립니다)

> 원고가 완벽하지 않아도 괜찮습니다. 텍스트만 있으면 구조화/편집은 제가 해드립니다.

## 10. 자주 묻는 질문 (FAQ)

**Q. 원고가 아직 완성이 안 됐는데 의뢰 가능한가요?**
A. 네! 80% 이상 완성된 원고면 충분합니다. 부분 원고도 상담 후 진행 가능합니다.

**Q. 영어 외 다른 과목도 가능한가요?**
A. 네, 수학/과학/국어/사회 등 모든 과목 가능합니다. 교재 디자인 시스템이 범용적입니다.

**Q. 인쇄소에 바로 넣을 수 있나요?**
A. 네, A4 인쇄 최적화 PDF로 납품하므로 인쇄소에 바로 입고 가능합니다.

**Q. 나중에 내용을 수정하고 싶으면요?**
A. 원본 파일 납품 옵션(2만원)을 선택하시면 직접 수정 가능합니다. 또는 수정 의뢰도 가능합니다.

**Q. 대량 교재도 가능한가요?**
A. 네! 자체 빌드 파이프라인으로 100페이지+ 교재도 제작 가능합니다. 무료 상담 후 별도 견적을 드립니다.

**Q. 전자책(e-book)으로도 만들 수 있나요?**
A. 네, 인쇄용 PDF뿐 아니라 웹 열람용 HTML 파일로도 납품 가능합니다.
