# 교재제작 영업 자산

이 디렉토리는 교재제작 서비스의 크몽/숨고 등록 자료와 이미지/스크립트를 모아둔 작업 폴더다.

## 구조
- `platform-listings/kmong/` — 크몽 등록 문안
- `platform-listings/soomgo/` — 숨고 등록 문안
- `assets/generated/` — 생성된 배너/슬라이드 이미지
- `assets/portfolio/` — 포트폴리오 이미지
- `assets/debug/` — 디버그 산출물
- `scripts/` — 생성/업로드/등록 스크립트
- `docs/` — README, 이미지 스펙, 프롬프트 문서
- `SOURCE_OF_TRUTH.md` — 이 폴더의 역할과 정본 규칙

## 주의
- LMS 영업 자료와 혼동하지 말 것
- 프로젝트 맵은 옵시디언 `복돌이/프로젝트-맵.md` 참조
