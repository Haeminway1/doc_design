# Kmong Launch Pack

이 폴더는 크몽 서비스 등록용 초안과 자동 생성 자산을 모아두는 용도다.

## 추천 운영 순서

1. 첫 등록은 `디자인 > 책표지·내지` 카테고리에서 메인 서비스 하나를 올린다.
2. 메인 서비스가 올라간 뒤 `문서·글쓰기 > 문서 편집` 카테고리로 저가 반복형 시험지/워크시트 편집 서비스를 분리한다.
3. 실제 거래 데이터를 본 뒤 내신 변형 워크북과 편입 모의고사는 별도 서비스로 독립시킨다.

## 첫 서비스 전략

- 제목: `영어 강사·학원용 내신 변형, 수능·편입 모의고사, 부교재를 고퀄리티로 제작해 드립니다`
- 포지션: 영어를 이해하는 교재 제작자
- 강점 메시지: 문제 퀄리티 + 디자인 퀄리티 + 학원형 결과물
- 카테고리 선택 이유:
  - `디자인 > 책표지·내지`에는 실제로 학원 교재 북 표지/내지/편집 디자인 서비스가 존재하고 44만 원 표지, 16.5만 원 내지 템플릿, 페이지당 조판 단가 구조가 확인된다.
  - `문서·글쓰기 > 문서 편집`은 교재/시험지 편집 수요가 있지만 가격대가 상대적으로 낮아 진입용 서브 서비스에 더 맞다.

## 가격 구조 메모

- STANDARD 129,000원: 8페이지 이하 시험지/워크시트/해설지 재편집
- DELUXE 390,000원: 24페이지 이하 부교재 제작
- PREMIUM 890,000원: 48페이지 이하 브랜드형 교재 패키지

## 업셀링 방향

- 내신 변형 워크북: 난이도 설계까지 포함하는 프리미엄 옵션
- 수능 실전모의고사: 실전형 레이아웃과 해설 구조로 차별화
- 편입 학교별 모의고사: 학교 유형 적합성을 전면에 내세운 블루오션 포지션
- 신규 자체교재: 가장 높은 객단가로 별도 협의
