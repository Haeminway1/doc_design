## 3) 쏠북 — 외주 서비스 확장 아님, 자료 판매 트랙으로 별도 설계 필요

### 공개 조사로 확인한 사실
- 저자 지원 페이지 존재: <https://promotion.solvook.com/apply/author>
- 요구 사항:
  - 과목/경력/활동계획 입력
  - 실제 자료 파일 1개 이상 제출
  - 판매 목적 완성본 + 문제/답지 포함 필요
  - 영업일 기준 5일 내 결과 안내

### 운영 해석
쏠북은 크몽/숨고처럼 ‘맞춤 외주 수주 플랫폼’이 아니라, ‘완성형 교육 자료 유통/판매 플랫폼’에 가깝다.

### 따라서 필요한 상품화 방향
- 내신 변형문제 세트
- 학교별 시험 대비 워크북
- 편입영어 독해/문법/어휘 자료
- 답지 포함 PDF 패키지
- 시리즈형 자료 카탈로그
