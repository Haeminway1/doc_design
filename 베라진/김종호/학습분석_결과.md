### 설명 A: -ing 핵심 정리

#### (1) "to"가 전치사인 표현 → 뒤에 반드시 -ing

> **판별법**: "to" 뒤에 **명사**를 넣어 봐라. 자연스러우면 전치사 to → **-ing**
> - "I look forward to **the party**" (자연스러움) → 전치사 to → -ing
> - "I want to **the party**" (부자연스러움) → to부정사 → 동사원형

| 표현 | 의미 | 예문 |
|------|------|------|
| **be used to** -ing | ~에 익숙하다 | I am used to **waking** up early. |
| **look forward to** -ing | ~를 기대하다 | I look forward to **seeing** you. |
| **be committed to** -ing | ~에 전념하다 | He is committed to **reducing** waste. |
| **be devoted to** -ing | ~에 헌신하다 | She is devoted to **teaching** kids. |
| **when it comes to** -ing | ~에 관해서는 | When it comes to **cooking**, she's the best. |
| **be subject to** -ing | ~의 대상이 되다 | The plan is subject to **changing**. |
| **be opposed to** -ing | ~에 반대하다 | They are opposed to **building** a dam. |
| **be prone to** -ing | ~하는 경향이 있다 | He is prone to **making** mistakes. |
| **be reduced to** -ing | ~하는 지경에 이르다 | She was reduced to **begging**. |
| **be limited to** -ing | ~에 한정되다 | Access is limited to **entering** the lobby. |
| **lead to** -ing | ~을 초래하다 | Stress can lead to **overeating**. |
| **resort to** -ing | ~에 의지하다 | He resorted to **borrowing** money. |
| **confess to** -ing | ~을 고백하다 | She confessed to **stealing** the ring. |
| **adjust to** -ing | ~에 적응하다 | I adjusted to **living** alone. |
| **take to** -ing | ~하기 시작하다 | She took to **jogging** every morning. |
| **get around to** -ing | 시간을 내어 ~하다 | I'll get around to **fixing** it. |
| **prefer A to** -ing | ~보다 선호하다 | I prefer reading to **watching** TV. |
| **with a view to** -ing | ~할 목적으로 | He studied with a view to **passing** the exam. |
| **in addition to** -ing | ~에 더하여 | In addition to **studying**, she works. |
| **object to** -ing | ~에 반대하다 | I object to **being** treated unfairly. |

---

#### (2) be used to -ing vs. used to V

| 구분 | **be used to -ing** | **used to + V** |
|------|---------------------|-----------------|
| 의미 | ~하는 것에 **익숙하다** | (과거에) ~하곤 **했다** |
| 시제 | 현재/과거 모두 가능 | **과거만** 가능 |
| to | **전치사** → -ing | **to부정사** → 동사원형 |
| 예문 | I **am used to** getting up early. | I **used to** get up early. |
| 해석 | 일찍 일어나는 것에 익숙하다 | 일찍 일어나곤 했다 (지금은 아님) |

**추가**: **get used to -ing** = ~에 익숙해지다 (변화 과정)

```
(X) I used to waking up early.        → used to + 동사원형이므로 wake
(X) I am used to wake up early.       → be used to + -ing이므로 waking
(O) I used to wake up early.          → 과거 습관
(O) I am used to waking up early.     → 현재 익숙함
```

---

#### (3) -ing 구별 5단계 (동명사 vs 현재분사)

```
[1단계] 주어 자리? → 동명사
   예: Swimming is fun.

[2단계] 타동사/전치사의 목적어? → 동명사
   예: I enjoy traveling. / good at solving

[3단계] be동사 뒤 보어?
   → S = C 동호관계 → 동명사  (His hobby is painting.)
   → S의 성질/감정 → 현재분사  (The movie was boring.)
   → 진행형 → 현재분사        (She is running.)

[4단계] 명사 수식?
   → 용도/목적 → 동명사       (a washing machine = 세탁용)
   → 진행 동작 → 현재분사     (the setting sun = 지고 있는)
   ※ "명사가 실제로 그 동작을 하는가?" YES→현재분사, NO→동명사

[5단계] 지각/사역동사 + O + -ing / 분사구문 → 현재분사
   예: I saw him running. / Walking through the park, she found...
```

**4단계 핵심 비교표** (실전 17번·19번에서 틀린 포인트):

| -ing + 명사 | 해석 | 구분 | 이유 |
|-----------|------|------|------|
| a **washing** machine | 세탁**용** 기계 | 동명사 | 기계가 세탁하고 있지 않음 |
| a **sleeping** bag | 잠자**기 위한** 가방 | 동명사 | 가방이 자고 있지 않음 |
| a **parking** lot | 주차**용** 공간 | 동명사 | 공간이 주차하고 있지 않음 |
| the **setting** sun | 지**고 있는** 태양 | 현재분사 | 태양이 실제로 지고 있음 |
| the **rising** sun | 떠오르**고 있는** 태양 | 현재분사 | 태양이 실제로 떠오르고 있음 |
| a **crying** baby | 울**고 있는** 아기 | 현재분사 | 아기가 실제로 울고 있음 |
| **boiling** water | 끓**고 있는** 물 | 현재분사 | 물이 실제로 끓고 있음 |

### 설명 B: There + be동사 = 1형식

**"There are many dishes in the cupboard."** 분석:
- There = **유도부사** (주어가 아님!)
- 진짜 주어 = many dishes
- are = **"있다"** (존재) → 1형식
- in the cupboard = 수식어(부사구)

| be동사의 의미 | 형식 | 예문 |
|-----------|------|------|
| **"있다" (존재)** | **1형식** | There **is** a cat on the roof. |
| **"~이다" (상태)** | **2형식** | The cat **is** cute. |

---

### 설명 C: 1형식 vs 2형식 완전 판별법

> POINT 1-1 연습문제 3, 9, 10, 11, 12, 13번에서 집중적으로 틀린 포인트

#### 핵심 구분

| | 1형식 | 2형식 |
|--|------|------|
| **구조** | S + V (+수식어) | S + V + **주격보어(C)** |
| **동사** | 완전자동사 | **연결동사** |
| **보어** | 없음 (수식어만) | 명사 또는 형용사 |
| **판별** | 동사 뒤를 빼도 문장 성립 | 보어를 빼면 문장 불완전 |

#### 연결동사 (2형식 동사) 목록

| 유형 | 동사 | 의미 |
|------|------|------|
| 상태 유지 | **be, stay, keep, remain** | ~이다, ~인 채로 있다 |
| 상태 변화 | **become, get, turn, grow, go** | ~이 되다, ~해지다 |
| 감각 | **look, sound, smell, taste, feel** | ~처럼 보이다/들리다/... |

#### 판별 3단계

```
[1단계] 동사가 연결동사 목록에 있는가?
   → NO → 1형식 (또는 3형식 이상)
   → YES → 2단계로

[2단계] 동사 뒤에 형용사 또는 명사(보어)가 있는가?
   → YES → 2형식
   → NO (부사/전치사구만 있음) → 1형식

[3단계] 보어를 빼면 의미가 불완전한가?
   → YES → 2형식 확정
   → NO → 1형식
```

#### 혼동하기 쉬운 경우 — 완전 정리

| 문장 | 형식 | 판별 근거 |
|------|------|---------|
| The snail moves **slowly**. | **1형식** | slowly = **부사** (보어 아님). move = 자동사 |
| The sky grew **dark**. | **2형식** | dark = **형용사**(보어). grew = 연결동사 "~해지다" |
| We must keep **quiet**. | **2형식** | quiet = **형용사**(보어). keep = 연결동사 "~인 채로 유지하다" |
| An accident **happened** yesterday. | **1형식** | happened = 자동사. yesterday = 부사(수식어) |
| Olivia became **a ballerina**. | **2형식** | a ballerina = **명사**(보어). became = 연결동사 |
| There **are** many dishes. | **1형식** | are = "있다"(존재). There = 유도부사 |
| The stars will **shine** brightly. | **1형식** | brightly = **부사** (보어 아님). shine = 자동사 |

> **핵심 함정**: **부사 vs 형용사** 구분이 가장 중요!
> - **slowly, brightly, quietly** = **부사** → 수식어 → **1형식**
> - **slow, bright, quiet** = **형용사** → 보어 가능 → **2형식**
>
> 예: "He moved **slowly**." (1형식) vs "He remained **slow**." (2형식)

### 문제 1: POINT 3-3 연습문제 6번

> **Let me (explain / to explain) the rules of this game.**

#### 정답: **explain**

#### 해설
- **let**은 사역동사 → 목적격 보어로 **동사원형**만 가능
- "Let me explain" ✅ / "Let me to explain" ❌

| 동사 | 보어 형태 | 예문 |
|------|----------|------|
| let | **동사원형** | Let me **explain**. |
| make | **동사원형** | She made me **clean**. |
| have | **동사원형** | I had him **fix** it. |
| allow | **to부정사** | Allow me **to explain**. |

#### 변형문제

**1.** My parents didn't let me (stay / to stay) out late.
> 정답: **stay** — let + O + 동사원형

**2.** The teacher let the students (choose / to choose) their own topics.
> 정답: **choose** — let + O + 동사원형

**3.** Please let me (know / to know) if you need any help.
> 정답: **know** — let + O + 동사원형

**4.** They wouldn't let anyone (enter / to enter) the building after 10 p.m.
> 정답: **enter** — let + O + 동사원형

**5.** 다음 중 어법상 올바른 문장은?
> ① She let him to borrow her pen.
> ② She allowed him borrow her pen.
> ③ She let him borrow her pen.
> ④ She let him borrowing her pen.
>
> 정답: **③** — let + O + 동사원형. ②는 allow + O + to V 필요.

### 문제 2: POINT 3-3 연습문제 8번

> **What made you (sleep / to sleep) so late yesterday?**

#### 정답: **sleep**

#### 해설
- **make** = 사역동사 → **동사원형**
- 단, **수동태**에서는 **to부정사**: "I was made **to sleep** early."

| 태 | 구조 | 예문 |
|---|------|------|
| 능동 | make + O + **동사원형** | The noise **made** me **wake** up. |
| 수동 | be made + **to V** | I **was made to wake** up. |

#### 변형문제

**1.** The sad movie made her (cry / to cry).
> 정답: **cry** — make + O + 동사원형

**2.** What makes this restaurant (stand / to stand) out?
> 정답: **stand** — make + O + 동사원형

**3.** The rain made us (cancel / to cancel) the event.
> 정답: **cancel** — make + O + 동사원형

**4.** 다음 중 어법상 **틀린** 것은?
> ① The teacher made the students study harder.
> ② The cold weather made him to wear a coat.
> ③ She made me wait for two hours.
> ④ The joke made everyone laugh.
>
> 정답: **②** — to wear → wear

**5.** He was made _______ the report again by his boss.
> ① rewrite  ② to rewrite  ③ rewriting  ④ rewrote
>
> 정답: **②** — make 수동태: be made + to V

### 문제 3: POINT 2 연습문제 B 3번

> **We made Sandra a birthday card.** → 3형식으로 전환

#### 정답: **We made a birthday card for Sandra.**

#### 해설
- make는 **for**를 쓰는 동사 (buy, cook, find, make, get, build)
- "전달" 느낌 → **to** (give, send, bring, pass, show, teach, tell)
- "대신/위해" 느낌 → **for** (make, buy, cook, find, get, build)

#### 변형문제

**1.** She bought her son a new backpack. → 3형식
> **She bought a new backpack for her son.** (buy → for)

**2.** The chef cooked us a special meal. → 3형식
> **The chef cooked a special meal for us.** (cook → for)

**3.** He found me a good seat. → 3형식
> **He found a good seat for me.** (find → for)

**4.** "My father built a treehouse _____ my brother."  ① to ② for ③ of ④ at
> 정답: **②** — build → for

**5.** 다음 중 3형식 전환이 올바른 것은?
> ① She gave a book for me.  ② He made a cake to his wife.
> ③ I sent a letter to my friend.  ④ Mom cooked a meal to us.
>
> 정답: **③** — send → to. ①give→to, ②make→for, ④cook→for

### 문제 4: 실전 30문항 5번

> **His biggest fear is failing the exam.** → failing: 동명사 or 현재분사?

#### 정답: **동명사**

#### 해설
- 주어(fear) = 보어(failing) → **동호관계** → 동명사
- "두려움 = 떨어지는 것" → 같은 대상

| S = C (동호관계) | → 동명사 | His fear is **failing**. |
|S의 성질/감정 설명 | → 현재분사 | The movie is **boring**. |

#### 변형문제

**1.** My favorite activity is hiking in the mountains. → hiking은?
> **동명사** — activity = hiking 동호관계

**2.** The news was surprising to everyone. → surprising은?
> **현재분사** — news의 성질 설명

**3.** Her dream is becoming a doctor. → becoming은?

### 문제 7: POINT 1-2 기출 적중문제

> **The story sounds a lie.** → 틀린 부분을 고치시오.

#### 정답: **The story sounds like a lie.**

#### 해설
- 감각동사 뒤에 **명사** → 반드시 **like + 명사**
- 감각동사 뒤에 **형용사** → 바로 가능

```
감각동사 + 형용사      → OK  (It sounds strange.)
감각동사 + like + 명사  → OK  (It sounds like a lie.)
감각동사 + 명사        → NO  (It sounds a lie. ❌)
감각동사 + 부사        → NO  (It sounds strangely. ❌)
```

#### 변형문제

**1.** 고치시오: "This soup tastes salt."
> **This soup tastes salty.** — salt(명사) → salty(형용사)

**2.** 올바른 문장은?
> ① The flower smells sweetly.  ② The flower smells sweet.
> ③ The flower smells like sweetly.  ④ The flower smells as sweet.
>
> 정답: **②** — 감각동사 + 형용사

**3.** "Her voice sounds _______ an angel."  ① as ② like ③ alike ④ likely
> 정답: **②** — 감각동사 + like + 명사

**4.** 고치시오: "The fabric feels like smooth."
> **The fabric feels smooth.** — 형용사 앞에 like 불필요

**5.** 어법상 틀린 것을 **모두** 고르시오.
> ① He looks happily today.  ② The milk smells like cheese.
> ③ This cake tastes like delicious.  ④ The idea sounds great.
>
> 정답: **①, ③** — ①happily→happy, ③like delicious→delicious

### 문제 8: 편입 기출 3번

> **She denied _______ anything about the missing documents.**
> (A) know  (B) to know  (C) knowing  (D) known

#### 정답: **(C) knowing**

#### 해설
- **deny**는 **동명사만** 목적어로 취하는 동사 → deny + V-ing
- "She denied **knowing** anything" = 그녀는 아무것도 모른다고 부인했다
- deny + to know (❌) — to부정사 불가!

**동명사만 목적어로 취하는 동사** (시험 필수 암기):

| 동사 | 의미 | 예문 |
|------|------|------|
| **enjoy** | 즐기다 | I enjoy **swimming**. |
| **mind** | 꺼리다 | Do you mind **waiting**? |
| **deny** | 부인하다 | He denied **stealing** the money. |
| **finish** | 끝내다 | She finished **writing** the report. |
| **avoid** | 피하다 | He avoids **making** eye contact. |
| **give up** | 포기하다 | She gave up **trying**. |
| **keep** | 계속하다 | He kept **talking**. |
| **suggest** | 제안하다 | I suggest **leaving** early. |
| **consider** | 고려하다 | Have you considered **moving**? |
| **practice** | 연습하다 | She practices **playing** the piano. |
| **postpone** | 연기하다 | They postponed **launching** the product. |
| **quit** | 그만두다 | He quit **smoking**. |

> **암기 팁**: **"mega" + 더** = mind, enjoy, give up, avoid + deny, finish, keep, suggest, consider...

#### 변형문제

**1.** The suspect denied (to commit / committing) the crime.

> ③ They considered moving to a new city.

### 문제 9: 편입 기출 4번

> **_______ from the top of the mountain, the city looked breathtaking.**
> (A) See  (B) Seeing  (C) Seen  (D) Having seeing

#### 정답: **(C) Seen**

#### 해설
- 이것은 **분사구문**이다.
- 주절의 주어 = **the city** (도시)
- 도시는 "보는" 주체가 아니라 "**보여지는**" 대상 → **수동** 관계
- 수동 → **과거분사(Seen)** 사용

> **핵심**: 분사구문에서 주어와 분사의 관계가 **능동**이면 **현재분사(-ing)**, **수동**이면 **과거분사(p.p.)**

| 관계 | 분사 | 예문 |
|------|------|------|
| **능동** (주어가 동작 수행) | **현재분사** | **Walking** through the park, **she** found a wallet. (그녀가 걷는 중) |
| **수동** (주어가 동작 당함) | **과거분사** | **Seen** from the top, **the city** looked beautiful. (도시가 보여진 것) |

**왜 (B) Seeing이 아닌가?**
- Seeing이면: "도시가 보고 있는" (능동) → 도시는 눈이 없으므로 볼 수 없다!
- Seen이면: "도시가 보여지는" (수동) → 산꼭대기에서 도시가 보여짐 ✅

#### 변형문제

**1.** _______ in simple English, the book is easy to understand.
> (A) Writing  (B) Written  (C) To write  (D) Having writing
>
> 정답: **(B) Written** — 책이 "쓰여진" 것 (수동) → 과거분사

**2.** _______ by the beautiful scenery, she took many photos.
> (A) Impressing  (B) Impressed  (C) To impress  (D) Having impressing
>
> 정답: **(B) Impressed** — 그녀가 "감명받은" 것 (수동) → 과거분사

**3.** _______ hard for months, he finally passed the exam.
> (A) Study  (B) Studied  (C) Having studied  (D) Having been studied
>
> 정답: **(C) Having studied** — 그가 "공부한" 것 (능동) → 현재분사 완료형

**4.** _______ from a distance, the mountain looks small.
> (A) Seeing  (B) Seen  (C) To see  (D) Having seen
>
> 정답: **(B) Seen** — 산이 "보여진" 것 (수동) → 과거분사

**5.** _______ with care, this machine will last for years.
> (A) Using  (B) Used  (C) To use  (D) Having using
>
> 정답: **(B) Used** — 기계가 "사용되는" 것 (수동) → 과거분사

### 문제 10: 편입 기출 8·9·10번 통합

> **8.** The government is committed to _____ air quality in urban areas.
> → (A) improve **(B) improving** (C) improved (D) improvement
>
> **9.** When it comes to _____ a healthy diet, many people find it challenging.
> → (A) maintain **(B) maintaining** (C) maintained (D) be maintained
>
> **10.** The professor devoted his entire career to _____ the effects of climate change.
> → (A) study **(B) studying** (C) studied (D) have studied

#### 정답: 모두 **(B) -ing형**

#### 해설
세 문제 모두 **같은 문법 포인트**: "to"가 **전치사**이므로 뒤에 **동명사(-ing)**가 온다.

| 문제 | 표현 | to의 정체 | 이유 |
|------|------|---------|------|
| 8번 | **be committed to** | 전치사 | ~에 전념하다 → to + -ing |
| 9번 | **when it comes to** | 전치사 | ~에 관해서는 → to + -ing |
| 10번 | **devote ... to** | 전치사 | ~에 헌신하다 → to + -ing |

> **함정**: "to" 뒤에 동사원형을 넣고 싶은 유혹! 하지만 이 표현들의 to는 **전치사**이므로 반드시 **-ing**

**빠른 판별**: "to" 자리에 "to **the project**"처럼 명사를 넣어 본다.
- "He is committed to **the project**" → 자연스러움 → 전치사 to → -ing ✅
- "He wants to **the project**" → 부자연스러움 → to부정사 → 동사원형

#### 변형문제

**1.** She is accustomed to (live / living) in a big city.
> 정답: **living** — be accustomed to + -ing (전치사 to)

**2.** I look forward to (hear / hearing) from you soon.
> 정답: **hearing** — look forward to + -ing

**3.** He objected to (change / changing) the schedule.
> 정답: **changing** — object to + -ing

**4.** In addition to (study / studying), she works part-time.
> 정답: **studying** — in addition to + -ing

**5.** 다음 중 어법상 **올바른** 것은?
> ① She is used to work under pressure.
> ② He devoted his life to help the poor.
> ③ When it comes to cook, she's an expert.
> ④ I'm looking forward to meeting you.

> 정답: **④** — look forward to + -ing. ①work→working, ②help→helping, ③cook→cooking

### 문제 11: POINT 1-1 연습 3·9·10·11·12·13번 통합

> **3번:** The snail moves slowly. → 1형식 / 2형식?
> **9번:** We must keep quiet in the hallway. → 1형식 / 2형식?
> **10번:** An accident happened yesterday. → 1형식 / 2형식?
> **11번:** Olivia became a ballerina. → 1형식 / 2형식?
> **12번:** There are many dishes in the cupboard. → 1형식 / 2형식?
> **13번:** The stars will shine brightly tonight. → 1형식 / 2형식?

#### 정답 및 해설

| # | 문장 | 정답 | 해설 |
|---|------|------|------|
| 3 | The snail moves **slowly**. | **1형식** | slowly = **부사**(수식어). 부사는 보어가 아님. move = 자동사 |
| 9 | We must keep **quiet**. | **2형식** | quiet = **형용사**(보어). keep = 연결동사 "~한 상태를 유지하다" |
| 10 | An accident **happened** yesterday. | **1형식** | happened = 완전자동사. yesterday = 부사(수식어) |
| 11 | Olivia became **a ballerina**. | **2형식** | a ballerina = **명사**(보어). became = 연결동사 "~이 되다" |
| 12 | There **are** many dishes in the cupboard. | **1형식** | There = 유도부사(주어 아님). are = "있다"(존재). 주어 = dishes |
| 13 | The stars will **shine** brightly. | **1형식** | brightly = **부사**(수식어). shine = 자동사 |

#### 오답 패턴 분석

**패턴 1: 부사를 보어로 착각** (3번, 13번)
- slow**ly**, bright**ly** → **-ly**로 끝나면 대부분 부사 → **보어 아님** → 1형식
- slow, bright → 형용사 → 보어 가능 → 2형식

**패턴 2: 연결동사를 일반동사로 착각** (9번, 11번)
- keep quiet: keep이 "유지하다"(연결동사)로 쓰임 → quiet(형용사) = 보어 → 2형식
- became a ballerina: became = 연결동사 → 2형식

**패턴 3: There + be동사 혼동** (12번)
- There is/are = "있다"(존재) → 1형식
- 주어 is/are + 보어 = "이다"(상태) → 2형식

#### 같은 동사, 다른 형식 — 비교

| 1형식 (자동사) | 2형식 (연결동사) |
|-----------|-----------|
| He **kept** walking. (3형식: 계속했다) | He **kept** silent. (2형식: 유지했다) |
| She **grew** flowers. (3형식: 키웠다) | She **grew** tall. (2형식: 키가 커졌다) |
| He **turned** left. (1형식: 방향을 틀었다) | He **turned** pale. (2형식: 창백해졌다) |
| She **got** a letter. (3형식: 받았다) | She **got** angry. (2형식: 화가 났다) |

#### 변형문제

**1.** The baby cried loudly all night. → 1형식 / 2형식?
> 정답: **1형식** — loudly = 부사(수식어). cried = 자동사

**2.** The milk went bad after three days. → 1형식 / 2형식?
> 정답: **2형식** — bad = 형용사(보어). went = 연결동사 "~해지다"

**3.** She remained calm during the crisis. → 1형식 / 2형식?
> 정답: **2형식** — calm = 형용사(보어). remained = 연결동사 "~한 상태로 남다"

**4.** The children ran quickly to the playground. → 1형식 / 2형식?
> 정답: **1형식** — quickly = 부사(수식어). ran = 자동사

**5.** 다음 중 **2형식** 문장을 **모두** 고르시오.
> ① The soup smells delicious.
> ② He walked slowly to the door.
> ③ She stayed healthy throughout her life.
> ④ The bird flew high in the sky.
> ⑤ The weather turned cold suddenly.
>
> 정답: **①, ③, ⑤**
> - ① smells + delicious(형용사) = 2형식
> - ② slowly(부사) = 1형식
> - ③ stayed + healthy(형용사) = 2형식
> - ④ high(부사적 용법) = 1형식
> - ⑤ turned + cold(형용사) = 2형식

## 총 요약 — 복습 체크리스트

### 사역동사
- [ ] make, have, let + O + **동사원형**
- [ ] make 수동태: be made + **to V**
- [ ] let ≠ allow: let + 동사원형 / allow + to V

### 4형식 → 3형식
- [ ] make, buy, cook, find, get, build → **for**
- [ ] give, send, bring, pass, show, teach, tell → **to**
- [ ] ask → **of**

### 감각동사
- [ ] 감각동사 + **형용사** (부사 ❌)
- [ ] 감각동사 + **like + 명사** (명사 단독 ❌)

### 동명사를 목적어로 취하는 동사
- [ ] deny, enjoy, mind, finish, avoid, give up, keep, suggest, consider, practice, postpone, quit + **V-ing**

### 분사구문 능동/수동
- [ ] 주어가 동작 **수행** → **현재분사 (V-ing)**
- [ ] 주어가 동작 **당함** → **과거분사 (p.p.)**

### 전치사 to + -ing
- [ ] be committed to, be devoted to, look forward to, be used to, when it comes to, object to, in addition to... + **-ing**

### 1형식 vs 2형식
- [ ] **부사**(-ly)는 보어가 아님 → **1형식**
- [ ] **형용사**는 보어 → **2형식**
- [ ] There + be = "있다" → **1형식**
- [ ] 연결동사(be, become, get, turn, grow, keep, stay, remain, 감각동사) + 보어 → **2형식**

### -ing 구별
- [ ] 주어/목적어 → **동명사**
- [ ] 보어: S=C → **동명사** / 성질 → **현재분사**
- [ ] 명사 수식: 용도 → **동명사** / 동작 → **현재분사**
- [ ] 지각·사역동사 O.C. / 분사구문 → **현재분사**
