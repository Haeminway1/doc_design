## 🔧 3. 기술 스택 & 구현 방법

### 방법 A: Puppeteer 완전 자동화 (이상적)

**장점:**
- 완전 자동화 (실행만 하면 끝)
- 반복 작업에 최적
- 스케줄링 가능 (cron)

**단점:**
- 카카오비즈니스 페이지 구조 복잡 (SPA, 동적 렌더링)
- DOM 타이밍 이슈
- 개발 시간 1-2시간 소요

**기술:**
- Node.js + Puppeteer
- CDP (Chrome DevTools Protocol) 연결

**핵심 과제:**
1. ❌ **채팅 목록 파싱:** `[role="grid"] [role="row"] [role="listitem"]` 구조 추출 실패
   - Grid/Row는 찾지만 listitem이 빈 배열로 반환됨
   - JavaScript 렌더링 타이밍 문제 추정
2. 채팅방 클릭 및 진입
3. 메시지 히스토리 스크롤
4. 이미지 요소 추출
5. 이미지 다운로드 (URL → 파일)

---

### 방법 B: 브라우저 DevTools + 수동 다운로드 (현실적, 추천)

**장점:**
- 즉시 사용 가능 (개발 시간 0)
- 안정적
- 이미지 품질 보장

**단점:**
- 반복 작업 시 수동 작업 필요
- 자동화 불가

**프로세스:**
1. 카카오비즈니스 채팅방 접속
2. F12 → Network 탭 → Img 필터
3. 채팅방별로 이미지 우클릭 → "다른 이름으로 저장"
4. 저장 경로: Downloads → 수동 정리 스크립트 실행

**보조 스크립트 (자동 정리):**
```bash
# organize-downloads.sh
# Downloads 폴더의 이미지를 학생별 폴더로 자동 분류
# 사용법: ./organize-downloads.sh 김지환 260215
```

---

### 방법 C: 브라우저 콘솔 JavaScript (반자동)

**장점:**
- 브라우저에서 바로 실행
- 이미지 URL 자동 추출 가능
- DevTools 없이 실행 가능

**단점:**
- 각 채팅방마다 실행 필요
- 다운로드는 여전히 수동/스크립트

**프로세스:**
1. 채팅방 열기
2. F12 → Console
3. 아래 스크립트 붙여넣기:
   ```javascript
   // 현재 채팅방의 모든 이미지 URL 추출
   const images = Array.from(document.querySelectorAll('img'))
     .filter(img => img.src.includes('kakao') || img.src.includes('phinf'))
     .map(img => img.src);
   
   console.log('이미지 URL:');
   images.forEach((url, i) => console.log(`${i+1}. ${url}`));
   
   // 복사 가능하게 JSON으로 출력
   copy(JSON.stringify(images, null, 2));
   ```
4. 복사된 URL 리스트 → wget/curl 스크립트로 다운로드

---

### 방법 D: Webhook 수신 (미래 확장)

**개념:**
- 카카오비즈니스 챗봇 연동
- 메시지 수신 시 실시간 웹훅
- 이미지 자동 저장

**장점:**
- 실시간 처리
- 완전 자동화
- 학생이 보낸 즉시 처리

**단점:**
- 챗봇 개발 필요
- 카카오 챗봇 관리자센터 설정 필요
- 월렛 연결 + 과금 (메시지당 15원)
- 과거 메시지 조회 불가 (실시간만)

**참고:**
- Event API 문서: https://kakaobusiness.gitbook.io/main/tool/chatbot/main_notions/event-api
- 웹훅은 "보내기" 전용, "받기"는 챗봇 스킬 서버 필요
