# 베라진 편입영어 학생관리 자동화 고도화 (V2)

**목표:** 멱등성(Idempotency)을 보장하는 견고한 데이터 수집 파이프라인 구축 및 AI 기반의 즉각적인 컨텍스트 분류 시스템 도입.

---

## 1. 멱등성 있는 다운로드 시스템 (Ingestion)

**문제점:**
- 기존 스크립트는 실행 시점마다 파일을 다운로드하여 중복 파일 발생 (낮/밤 실행 시 문제).
- 파일명만으로 중복 여부를 판단하기 어려움.

**해결 방안:**
- **고유 ID 활용:** 카카오톡 메시지 DOM의 `data-msg-id` 또는 이미지 URL의 고유 해시를 파일명에 포함.
  - 포맷: `[YYMMDD]_[MSG_ID]_[SEQ].[ext]`
- **중복 방지 로직:**
  1. 다운로드 전 대상 폴더 스캔 (`ls`).
  2. 파일명에 동일한 `MSG_ID`가 존재하면 다운로드 스킵 (API/네트워크 비용 절약).
  3. 존재하지 않는 경우에만 다운로드 수행.
- **PDF 지원:**
  - 이미지(`img`) 태그뿐만 아니라 파일 첨부(`a` 태그, 파일 아이콘)도 감지하여 PDF 다운로드 지원.
  - PDF는 원본 유지 (PNG 변환은 필요 시 썸네일 용도로만 제한적으로 수행).

## 2. 실시간 AI 분류 및 메타데이터 생성 (Classification)

**목표:** 다운로드 즉시 파일의 성격을 파악하고 구조화된 데이터로 기록.

**프로세스:**
1. **Trigger:** 다운로드 완료 직후 (또는 배치 실행 직후).
2. **Analysis:** 가벼운 Vision 모델(Gemini Flash 등)을 사용하여 이미지/PDF 첫 페이지 분석.
3. **Classification Categories:**
   - 📝 **플래너/일지:** 학습 계획표, 공부 시간 기록.
   - ✅ **인증/타임랩스:** 공부하는 모습, 책상 사진 등 단순 인증.
   - ❓ **질문/오답:** 문제집 사진, 모르는 문장 질문 (피드백지 제작 대상).
   - 💬 **기타:** 잡담, 공지, 의미 없는 이미지.
4. **Metadata Generation:**
   - 해당 날짜 폴더에 `_summary.md` (또는 `.json`) 자동 생성/갱신.

**출력 예시 (`_summary.md`):**
```markdown
## 2026-02-24 (학생명)

### 📊 요약
- **플래너:** ✅ 제출됨 (20:30)
- **피드백 필요:** 2건 (독해, 논리)

### 📂 파일 상세
| 파일명 | 분류 | 내용 요약 | 처리상태 |
|---|---|---|---|
| `260224_msg123_01.jpg` | 📝 플래너 | 8H 30M 달성 | 완료 |
| `260224_msg124_01.jpg` | ❓ 질문 | 논리 15번 오답 이유 질문 | **[대기중]** |
| `260224_msg125_01.pdf` | 📄 자료 | 위클리 테스트지 | 완료 |
```

## 3. 실행 전략 (Execution)

- **Script (Node.js):**
  - 브라우저 제어 (Puppeteer/CDP) 및 DOM 파싱.
  - 파일 다운로드 및 중복 체크 로직.
  - Markdown 파일 생성 및 관리.
- **AI (LLM):**
  - 이미지 비전 분석 및 분류 판단.
  - `_summary.md` 내용 생성.
